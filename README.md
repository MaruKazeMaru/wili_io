## WiLIについて
なくしもの捜索用ROS2パッケージ群です。<br>
名前はWhere is a Lost Itemの頭文字から取りました。

## 概要
以下の機能を含むROS2パッケージです。
- WiLI-外部アプリケーション間で通信
- WiLIの諸パラメータをデータベースで管理

データベースソフトウェアにはSQLite3を利用しています。<br>
言語はPythonです。<br>
[wili_db](http://gihub.com/MaruKazeMaru/wili_db)から派生しました。
